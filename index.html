    Blockchain Secret Phrases Hacking and Tracker

    A useful tool created by myself in order to provide a secure way to find a missing word in your secret phrase. Block chain is known for its security, but it is catastrophic if you unfortnately forget your secret phrase. But it is still able to crack if only one word or maybe 2 is missing. The tool is running on solana network based on bip-39 standards for 12 words secret phrase. Ideally, it could run with bip-39 standards of 12 and 24 words in any network with a small change. In my example, I am playing with 12 words and assume that you miss one word in any of position of your secret phrase. So you provide 11 words, run the hacking scripts, and wait some time, probably you go to sleep and wait for tomorrow, all the possible valid combinations would show. After that, you run another tracker scripts, and it tracks the solana native token balance with a few minutes, and output as a excel file, so you will be able to look at balance of each address in a straight forward manner. It is able to track any balance of the solana native token within that address by default, but you can track other SPL tokens with setting the token address in the configuration file. I know there could be some 3rd party online tool that does this for you, probably faster, better, and supports more networks. But would you trust some other 3rd party online tools by providing your secret phrase to them, and they can steal your money within a second? I don't trust that at all, however, a transparent script working in the offline mode definitely help you better with an secure approach.
